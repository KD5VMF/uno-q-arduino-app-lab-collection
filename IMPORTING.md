# Importing an App into Arduino App Lab

## Quick steps

1. Launch **Arduino App Lab**
2. Open **My Apps**
3. Select **Create new app +**
4. Choose **Import App**
5. Pick a project zip from `apps/` (example: `Pong Matrix AI.zip`)
6. Wait for the import to finish — the project should appear under **My Apps**

## Common mistakes

- **Unzipping first**: don't. Import the zip file directly.
- **Renaming internal files**: only rename the outer zip if you want; don't modify the contents unless you know what you're doing.
- **Wrong app**: if you have multiple Arduino tools installed, make sure you opened *Arduino App Lab* (not the IDE).

## What to share when asking for help

- Arduino App Lab version (shown in the title bar)
- Which zip you tried to import
- Any error message / screenshot
